实现字数统计和阅读时长功能

  1. 博客根目录运行git输入命令行
    npm install hexo-symbols-count-time --save

  2. 打开博客根目录，在最下方添加
    symbols_count_time:

    symbols: true
time: true
total_symbols: true
total_time: true

    参考自大佬LixT

增加Valine评论功能

  1. 进入leancloud官网注册登录控制台

  2. 创建一个应用，名字随便写。找到appkey中的appID和appkey

  3. 在主题配置文件中找到Valine
    valine:

    enable: false  #false改成true
appid:   #复制粘贴应用中的appid
appkey:  #复制粘贴应用中的appkey

其它的默认即可

修改文章底部带#的标签

  1. 用文本编辑器打开post.swig。路径themes\next\layout_macro\post.swig

  2. 找到
    rel="tags">#</a>
    这部分代码

  3. 将#替换成<i class="fa fa-tags"><i/>。注意空格

    想要什么icon就去fontawesome找。只要把相应图标的英文替换掉tags即可

4.重新hexo s刷新页面

实现Daovoice在线聊天功能

  1. 进入Daovoice官网注册登录。

  2. 进到Daovoice面板，点击左侧边栏的应用设置-- 安装到网站。在下方的代码中会看到app_id: "xxxx"字样。

  3. 打开 themes/next/layout/_partials/head/head.swig 文件中最下方加入如下代码：

在主题配置文件_config.yml，添加如下代码：

# DaoVoice 
daovoice: true
daovoice_app_id: 这里输入前面获取的app_id
  1. 回到Daovoice控制面板，点击聊天设置可以对聊天图标的颜色及位置进行设置。

此功能设置摘自#LixT#。

Deployer not found: git

博客根目录打开运行窗口，输入

npm install –save hexo-deployer-git
